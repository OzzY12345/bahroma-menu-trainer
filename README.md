# BAHROMA Menu Trainer

Локальное PWA-приложение для изучения меню BAHROMA за 14 дней.

## Что внутри

- 211 карточек меню из `public/data/menu.json`
- дневной 14-дневный план
- режимы `Название -> состав` и `Состав -> название`
- оценки `Не знаю / Почти / Знаю`
- список ошибок
- сценарии официанта: мясное, без рыбы, сырное, неострое
- прогресс в `localStorage`

## Запуск на Windows

```powershell
npm start
```

Открыть:

```text
http://127.0.0.1:4173/
```

## Проверки

```powershell
npm test
node --check src/app.js
node --check src/learning.js
node --check scripts/serve.mjs
```

## Деплой

Приложение статическое. Для VPS достаточно отдать корень проекта через Nginx или скопировать файлы в отдельную директорию и настроить `root`/`alias`.
